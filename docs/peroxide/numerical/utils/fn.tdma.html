TriDiagonal Matrix Algorithm (TDMA)

Description

Solve tri-diagonal matrix system efficiently (O(n))

|b0 c0         | |x0|   |y0|
|a1 b1 c1      | |x1|   |y1|
|   a2 b2 c2   | |x2| = |y2|
|      ...     | |..|   |..|
|         am bm| |xm|   |ym|

Caution

You should apply boundary condition yourself
